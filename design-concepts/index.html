🍚 오늘 뭐 해먹지 — 디자인 컨셉 5종

가로로 스크롤해서 비교하세요. 레이아웃(베이스)은 동일하고 디자인 요소만 다릅니다.

1. 토스 리파인드

지금 스타일 유지 + 그라디언트 버튼, 깊어진 그림자, 이모지 아바타, 인사말. 가장 안전한 업그레이드.

2. 웜 어피타이트

크림 배경 + 코랄 오렌지. 식욕을 돋우는 따뜻한 홈쿠킹 감성. 음식 앱다운 무드.

3. 프레시 글래스

파스텔 그라디언트 배경 + 반투명 유리(글래스모피즘) 카드 + 블루→민트 그라디언트. 산뜻하고 트렌디.

4. 다크 다이닝

딥 차콜 + 앰버 골드. 저녁 식탁 조명 같은 고급스러운 무드. 밤에 보기 편함.

5. 플레이풀 팝

두꺼운 라인 + 하드 섀도 + 스티커 뱃지 + "메뉴 뽑기" 컨셉. 게임처럼 재미있는 무드.
